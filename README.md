# Smart Label Project

This project is a **Smart Label System** that converts physical product labels into digital labels displayed on LED or LCD screens.

## Features
- **Backend:** Python + Flask
- **Frontend:** Web-based or device interface
- **Display Support:** LED and LCD screens
- **Device Integration:** Works with store display devices

## How to Run
1. Install dependencies: `pip install -r backend/requirements.txt`
2. Run the backend: `python backend/main.py`
3. Access the JSON output at: `http://localhost:5000/`
